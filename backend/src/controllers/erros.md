Respostas de erro do cliente (400 – 499)
400 Bad Request: A requisição está malformada ou o servidor não conseguiu entendê-la.

401 Unauthorized: O recurso exige autenticação (login) para ser acessado.

403 Forbidden: O cliente foi autenticado, mas não possui permissões de acesso ao recurso.

404 Not Found: A página ou o recurso solicitado não foi encontrado no servidor.

405 Method Not Allowed: O método HTTP utilizado (como GET ou POST) não é permitido no endpoint.

408 Request Timeout: O servidor desconectou porque a requisição demorou muito para ser concluída.

409 Conflict: Indica um conflito com o estado atual do recurso no servidor.

410 Gone: O recurso existia, mas foi deletado permanentemente e não possui redirecionamento.

422 Unprocessable Entity: A requisição está correta, mas contém erros de validação lógica nos dados.

429 Too Many Requests: O cliente enviou solicitações demais em um curto período (limite de taxa).
